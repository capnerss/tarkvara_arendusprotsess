Bitcoin kalkulaator.

Programmi eesmärk on teisendada Bitcoini (BTC) kogus teistesse valuutadesse, nagu EUR, USD, GBP, või EEK, kasutades reaalajas või hiljutist vahetuskurssi.

Programmi lühikirjeldus

Programm töötab järgmiste põhietappide kaudu:
Start ja Sisendi kogumine:
Programmi alustades küsitakse kasutajalt, kas ta on sisestanud Bitcoini koguse.
Kui ei, palutakse tal see sisestada ja protsess algab uuesti.
Järgmiseks küsitakse, kas ta on valinud sihtvaluuta.
Kui ei, palutakse tal valida valuuta.
Lõpuks küsitakse, kas kasutaja on vajutanud nuppu väärtuse arvutamiseks.
Valuuta valik ja API päringud:
Kui kõik eeldused on täidetud (kogus sisestatud, valuuta valitud, nuppu vajutatud), kontrollib programm, milline valuuta valiti (EUR, USD, GBP, või EEK).
Vastavalt valikule esitatakse API päring, et saada BTC hetkehind valitud valuutas (nt. EUR, USD, GBP).
Erijuhtum on EEK (Eesti kroon), mille puhul esitatakse API päring BTC hinna saamiseks EUR-is ja seejärel korrutatakse tulemus EEK-i hinnaga (ilmselt fiksseeritud kursi alusel, kuna EEK ei ole enam kasutusel).
Arvutamine ja Tulemus:
Pärast API-st kursi saamist korrutatakse sisestatud BTC kogus hetkehinnaga valitud valuutas.
Kontrollitakse, kas arvutus oli edukas (JAH või EI).
Edukal arvutusel kuvatakse tulemus aknas ja programm lõpetab töö (Lõpp).
Ebaõnnestunud arvutusel kuvatakse kasutajale veateade ("Tulemuse arvutus ebaõnnestus, palun proovi uuesti") ja protsess suunatakse tagasi algusesse (START).
See programm on tüüpiline näide valuutakalkulaatorist või konverterist, mis kasutab välist teenust (API) andmete saamiseks.
