Järjend (List)

Mis on järjend?

  • Järjend on muudetav (mutable) andmetüüp.
  • Võimaldab hoida mitut väärtust ühes muutujas.
  • Elemendid on järjestatud ja neil on indeks.
  • Kirjeldatakse nurksulgude abil: [].
  • Järjendis võib olla erineva andmetüübiga elemente (arvud, sõned, teised listid).

Järjendi loomine

empty_list = []
numbers = [1, 2, 5, 10]
names = ["Ago", "Kati", "Mati"]
mixed = [1, "Python", 3.14, [1, 2]]

Indekseerimine

Sarnaselt sõnele algavad indeksid 0-st.

  • names[0] -> "Ago"
  • names[-1] -> "Mati" (viimane element)
  • Elemendi muutmine: names[1] = "Piret"

Elementide lisamine ja eemaldamine

  • append(element) - lisab elemendi listi lõppu.
  • insert(index, element) - lisab elemendi määratud positsioonile.
  • extend(other_list) - ühendab teise listi elemendid olemasoleva listiga.
  • pop(index) - eemaldab ja tagastab elemendi (vaikimisi viimase).
  • remove(väärtus) - eemaldab esimese ettejuhtuva väärtuse.

Järjendi läbimine (for-tsükkel)

Kõige tavalisem viis listi sisu töötlemiseks:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Sorteerimine ja pikkus

  • len(list) - tagastab elementide arvu.
  • sort() - sorteerib listi elemendid (muudab algset listi).
  • sorted(list) - tagastab uue sorteeritud listi, algne jääb samaks.
  • reverse() - pöörab järjekorra tagurpidi.

Levinud operatsioonid

  • + - ühendab kaks listi.
  • * - kordab listi sisu.
  • in - kontrollib, kas element eksisteerib listis (tagastab True/False).
if "apple" in fruits:
    print("Olemas!")

List comprehension (Listmõistmine)

Lühike viis uue listi loomiseks olemasoleva põhjal:

numbers = [1, 2, 3, 4]
squares = [x**2 for x in numbers]
# tulemus: [1, 4, 9, 16]

Kokkuvõte

  • List on muudetav andmekogum.
  • Elemendid on järjestatud ja ligipääsetavad indeksi kaudu.
  • Võimaldab dünaamiliselt suurust muuta (lisada/eemaldada).
  • Sisu saab mugavalt läbida tsüklitega.
