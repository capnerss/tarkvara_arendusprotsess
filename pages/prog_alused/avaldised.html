Tehted arvudega (Computing with numbers)

Andmetüübid

  • Protsessor opereerib arvudega. Kõik andmetüübid esitatakse lõpuks arvudena (0 ja 1 jadana).
  • Programm käitub täis- ja murdarvudega erinevalt.
  • Pythonis on väärtusel andmetüüp, kuid muutujal endal andmetüüpi pole.
  • Funktsioon type() tagastab väärtuse andmetüübi.
print(type(3)) # <class 'int'>
print(type(3.14)) # <class 'float'>
print(type(3.0)) # <class 'float'>

Kuigi 3.0 on matemaatiliselt sama mis 3, on nende andmetüübid erinevad.

Ujukomaarvud (float)

  • Ujukomaarvud on ebatäpsed.
  • Näide: 0.1 + 0.1 + 0.1 annab tulemuseks 0.30000000000000004.
  • Võimalusel kasuta täisarve (nt hindade puhul 4.30 euro asemel 430 senti).

Tehted arvudega

  • + liitmine
  • - lahutamine
  • * korrutamine
  • / jagamine (tulemus on alati float)
  • ** astendamine
  • abs() absoluutväärtus
  • // täisarvuline jagamine
  • % jääk

Teisendamine

  • Kui avaldis sisaldab ujukomaarvu, teisendatakse kõik täisarvud ujukomaarvudeks.
  • int() - teisendab täisarvuks (murdosa kaotatakse, nt int(4.9) => 4).
  • float() - teisendab ujukomaarvuks.
  • Saab teisendada ka korrektse sisuga sõnesid: int("12") => 12.

Täisarvuline jagamine ja jääk

  • a // b - näitab, mitu korda jagaja jagatava sisse mahub (10 // 3 => 3).
  • a % b - näitab, palju jääb jagamisel üle (10 % 3 => 1).

Matemaatika moodul (math)

Täiendavate funktsioonide jaoks tuleb moodul importida: import math

  • math.pi - pii väärtus
  • math.sqrt() - ruutjuur
  • Sisaldab ka trigonomeetriat, logaritme ja ümardamist.

Tehete järjekord

  1. Astendamine
  2. Korrutamine, jagamine
  3. Liitmine, lahutamine

Sulgudega saab järjekorda muuta: (2 + 3) * 4 = 20.

Kokkuvõte

  • Väärtustel on andmetüüp (peamiselt int ja float).
  • Operatsioonide tulemus sõltub andmetüübist.
  • Tüüpe saab teisendada funktsioonidega int() ja float().
  • Keerukamad arvutused on kättesaadavad math moodulis.
