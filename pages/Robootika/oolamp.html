Ülesanne: Öölamp

Töö kirjeldus:

Nelja töörežiimiga öövalgusti fotoresistoriga:

  1. Režiim 1: Kõik LED-id välja lülitatud.
  2. Režiim 2: Ainult üks kahest valgusdioodist on sisse lülitatud (sõltuvalt valgustusest).
  3. Režiim 3: Kõik kaks dioodi on sisse lülitatud (sõltuvalt valgustusest).
  4. Režiim 4: Kõik 2 dioodi töötavad täisvõimsusel sõltumata fotoresistori valgustusest!

Kasutatud komponendid:

  • Arduino UNO plaat (1tk)
  • Arendusplaat (1tk)
  • Juhtmed (10tk)
  • Takisti (3tk, 220 Oomi ja 10000 Oomi)
  • LED (2tk: 1 valge ja 1 roheline)
  • Fotoresistor (1tk)
  • Potentsiomeeter (1tk)

Töö protsess ja LED-ide rakendamine:

Öövalgusti, millel on 4 erinevat töörežiimi, mida saab lülitada potentsiomeetri abil ja mis töötab fotoresistori ja valgusallikaga. Valgustus kodus või õues inimeste mugavuse huvides. Valgustub sõltuvalt ümbritsevast valguse hulgast.

Simulatsioon:

Kasuta potentsiomeetri liugurit režiimide vahetamiseks ja fotoresistori liugurit valgustaseme muutmiseks (pime vs valge).

Öölamp (LED-id)

Režiim: 1

Potentsiomeeter (A1): 0

Valgustus: Pime

Fotoresistor (A0): 0

Programmi kood (Arduino C++):

// Viigud
const int ldrPin = A0;
const int potPin = A1;

// 2 LED-i: valge ja roheline
const int ledPins[] = {9, 10};
const int ledCount = 2;

void setup() {
  for (int i = 0; i < ledCount; i++) {
    pinMode(ledPins[i], OUTPUT);
    digitalWrite(ledPins[i], LOW);
  }

  Serial.begin(9600);
}

void loop() {
  int lightLevel = analogRead(ldrPin);
  int potValue = analogRead(potPin);

  // Potentsiomeetri väärtus 4 tsooni kohta
  int mode = map(potValue, 0, 1023, 1, 5);
  if (mode > 4) mode = 4; // Kaitse maksimaalse väärtuse jaoks

  Serial.print("Režiim: ");
  Serial.print(mode);
  Serial.print(" | Valgustus: ");
  Serial.println(lightLevel);

  switch (mode) {
    case 1: // Kõik välja lülitatud
      controlLeds(0);
      break;

    case 2: // Üks LED sõltuvalt valgustusest
      if (lightLevel < 500) controlLeds(1);
      else controlLeds(0);
      break;

    case 3: // Kaks valgusdioodi sõltuvalt valgustusest
      if (lightLevel < 500) controlLeds(2);
      else controlLeds(0);
      break;

    case 4: // Kõik alati sees, olenemata valgusest
      controlLeds(2);
      break;
  }

  delay(100);
}

void controlLeds(int numberOn) {
  for (int i = 0; i < ledCount; i++) {
    digitalWrite(ledPins[i], i < numberOn ? HIGH : LOW);
  }
}
